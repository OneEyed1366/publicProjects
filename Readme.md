Проект: Кошелёк.ру (SPA приложения для получения данных по REST API)
Язык/Framework: Vue.js (Vuex + Axios)
Время выполнения: 5.47 ч.
Что было нужно сделать: SPA web-приложение для получения данных фондовой биржи по REST API с использование глобальной шины данных и преобразованием данных на клиенте, также было необходимо принудительно изменять Валюту для получения и в режиме реального времени отображать полученную информацию
Реализованные функции:
Адаптивность: Flexbox
Глобальная шина состояний: Vuex
Обращения к REST API: Axios
